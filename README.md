'코드'


| 명령어 | 기능 | 1 | 2 | 3 | 4 | 5 |
| :---: | :--- | :--- | :--- | :--- | :--- | :--- |
| top | 명령어는 시스템의 실시간 상태를 확인 | -d [초] : 업데이트 간격을 초 단위로 지정 | k : 특정 PID에 kill 시그널 보냄 | P : CPU 사용률 기준으로 정렬 (기본값) | M : 메모리 사용량 기준으로 정렬 | 엠티 |
| ps | 명령을 실행한 순간의 프로세스 상태를 정적으로(스냅샷) 보여줌 | a :	터미널과 관계된 모든 프로세스 표시 | u	: 소유자User 기반 상세 포맷으로 출력 | x :	터미널과 관계없는 프로세스도 모두 표시 | f : 프로세스 트리를 아스키 아트로 표현 | e :	모든 환경 변수를 포함하여 출력 |
| jobs | 현재 셸 세션 내에서 백그라운드 또는 정지 상태로 실행되고 있는 작업들의 목록과 상태를 보여줌 | Running : 백그라운드에서 실행 중 | Stopped : 실행이 일시 중지된 상태 | jobs -l : PID를 함께 출력 | jobs -p : PID만 출력 | 엠티 |
| kill | 지정된 프로세스 ID 또는 작업 번호에 시그널을 보내 프로세스를 종료하거나 제어함 | 시그널 1번 HUP(SIGHUP)	설정 파일 재로드 등 (데몬 재시작에 활용) | 시그널 2번	INT(SIGINT)	인터럽트 시그널, 일반적인 종료 요청 | 시그널 9번	KILL(SIGKILL)	강제 종료 시그널, 프로세스 무시 불가 | 시그널 15번	TERM(SIGTERM)	정상 종료 시그널, 프로세스 종료 전 작업 수행 가능 | 엠티 |
